Projeto: Lasfera RPG App

Stack:
- Next.js
- TypeScript
- Prisma
- SQLite
- Tailwind
- Componentes dentro de src/
- Não criar arquivos fora da arquitetura existente sem justificar

Regras:
- Não quebrar rotas existentes
- Não renomear pastas dinâmicas sem necessidade
- Usar componentes pequenos e reutilizáveis
- Separar lógica de jogo da interface
- Criar checkpoints claros
- Priorizar código simples, funcional e fácil de manter
